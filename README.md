Zomato Data Analysis
roject Ovrview:
This project analyzes Zomato restaurant data to identify pricing trends, rating distribution, and geographic performance across different countries and cities. The objective is to extract meaningful insights that can help improve pricing strategy, customer satisfaction, and business expansion decisions.

Zomato operates across multiple cities and countries with thousands of restaurants. The business needs insights into:
Distribution of restaurants by country and city
Relationship between pricing and ratings
Identification of high-performing and low-performing regions
Customer preference trends based on cost and ratings

Tools & Technologies Used:
Excel – Data validation and exploratory analysis
SQL – Data querying and aggregation
Power BI – Interactive dashboard development
Tableau – Data visualization and reportin

Data Cleaning & Preparation:
Handled missing and inconsistent values
Removed duplicate records
Converted data types for accurate analysis
Created calculated columns such as price buckets and rating categories

Key Analysis Performed:
Count of restaurants by country and city
Rating distribution analysis
Price bucket segmentation (Budget, Moderate, Premium, Luxury)
Relationship between average cost and ratings
KPI tracking for restaurant performance

Dashboard Features:
Interactive country-level map
Rating-based restaurant count visualization
Price bucket distribution chart
Filters for country, city, and rating
KPI cards for quick performance insights
